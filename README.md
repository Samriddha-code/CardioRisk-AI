# ❤️ CardioRisk AI

An AI-powered web application for predicting the risk of cardiovascular disease using machine learning.

This project uses a recall-optimized XGBoost model trained on a real-world cardiovascular disease dataset and deployed with Streamlit.

---

## 📌 Project Overview

Cardiovascular disease is one of the leading causes of death worldwide. Early identification of individuals at high risk can help support timely medical evaluation and intervention.

This application predicts whether a patient is at risk of cardiovascular disease based on demographic information, blood pressure, laboratory values, and lifestyle factors.

The model has been optimized to prioritize **Recall (Sensitivity)**, reducing the number of missed positive cases (False Negatives), which is especially important in medical screening applications.

## 📊 Final Model

**Algorithm**

- Recall-Tuned XGBoost Classifier

**Decision Threshold**

- 0.46

**Performance**

| Metric | Value |
|--------|------:|
| Accuracy | 72.33% |
| Precision | 69.65% |
| Recall | 78.09% |
| F1 Score | 73.63% |
| ROC-AUC | 80.02% |

The decision threshold was optimized to improve recall while maintaining a reasonable balance between precision and false positives.
